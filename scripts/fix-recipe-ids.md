### fix-recipe-ids.js

Convertit tous les id du fichier recipes.json en nombres (au cas où certains seraient des chaînes de caractères)

Vérifier que recipes.json est bien dans le dossier data/

```bash
node scripts/fix-recipe-ids.js
```
